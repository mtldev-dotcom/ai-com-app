https://developers.cjdropshipping.com/api2.0/v1/authentication/getAccessToken

```

                --header 'Content-Type: application/json' \
                --data-raw '{
                    "apiKey": "CJUserNum@api@xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
```
